# 卓越工程师大冒险 - 开发工作记录

## 项目概述
"卓越工程师大冒险"是一款像素风格的横版动作冒险游戏，包含腾讯大堂和小鹏汽车两个主题关卡。

## 已完成工作清单

### 1. 平台瓦片系统 (platform_tiles.png)
**文件**: [AssetLoader.js](file:///c:/Users/ashin/Downloads/卓越工程师大冒险/js/engine/AssetLoader.js)
- 实现了 `drawTilePlatform()` 方法，支持使用 `platform_tiles.png` 搭建平台和建筑
- 明确定义了三种平台类型的瓦片坐标：
  - **浮动平台 (platform)**: 左、中、右、单块四种样式
  - **地面平台 (ground)**: 左、中、右三种样式
  - **单向平台 (semisolid)**: 左、中、右三种样式
- 实现了瓦片拼接逻辑，支持任意宽度的平台渲染
- 添加了 fallback 渲染，当图片未加载时使用 canvas 绘制占位平台

### 2. 无缝背景滚动系统
**文件**: [TencentLobbyScene.js](file:///c:/Users/ashin/Downloads/卓越工程师大冒险/js/scenes/TencentLobbyScene.js)

- 背景图片水平平铺，形成无限滚动效果
- 视差因子 0.5，背景移动速度慢于前景，产生深度感
- 区域间平滑过渡：当玩家接近下一个区域时，背景逐渐混合
- 支持多个背景区域（腾讯大堂 → 微信区 → QQ区 → 游戏区 → 云与智慧产业区 → 平台内容区 → 技术工程区）

### 4. 装饰性 vs 伤害性障碍物区分
**文件**: [config.js](file:///c:/Users/ashin/Downloads/卓越工程师大冒险/js/config.js)
- 明确分离了两种障碍物类型：
  - **装饰性障碍物 (DECORATIVE_OBSTACLES)**: 17种，仅作视觉装饰，不造成伤害
    - 路障、雪糕筒、闸门、拱门、玻璃墙、冰柜、新闻屏、管道、墙壁等
  - **伤害性障碍物 (DEADLY_OBSTACLES)**: 3种，触碰即死
    - 红色电缆 (OBS_CABLE_RED)
    - 激光球 (OBS_LASER_SPHERE)
    - 损坏的充电器 (OBS_CHARGER_DAMAGED)
- 危险障碍物渲染时带有红色发光效果，视觉上有警示作用

## 技术要点

### 瓦片系统
- 使用非均匀瓦片坐标，精确匹配 platform_tiles.png 的图像结构
- 支持左、中、右三段式拼接，可渲染任意宽度的平台
- 三种平台类型：浮动平台、地面平台、单向平台

### 背景系统
- 视差滚动：背景移动速度 = 相机移动速度 × 视差因子
- 无缝平铺：背景图片水平重复排列
- 平滑过渡：区域间使用 alpha 混合，避免硬切

### AI 系统
- 状态机：巡逻 / 追踪两种状态
- 检测范围：圆形检测区域
- 跳跃逻辑：周期性跳跃，增加挑战性
- 速度平衡：病毒速度慢于玩家，玩家可通过走位甩开

### 碰撞系统
- AABB 碰撞检测
- 平台单向碰撞（从上方落下才碰撞）
- 玩家碰撞盒收缩（更精确的碰撞判定）
