# 《卓越工程师的大冒险》2D HTML RPG跑酷游戏 — 完整开发计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个完整的2D像素风HTML5 RPG跑酷游戏，包含腾讯主题解密关卡和小鹏主题生存跑酷关卡，使用Canvas渲染。

**Architecture:** 基于Canvas 2D的游戏引擎架构，采用场景管理器（SceneManager）驱动游戏流程，实体组件系统管理玩家/障碍/道具，AABB碰撞检测，localStorage存档。画布基准分辨率1920×1080，支持响应式缩放和移动端触控适配。

**Tech Stack:** HTML5 Canvas 2D, 原生JavaScript (ES6+), CSS3, localStorage, Web Audio API

```javascript
class GameLoop {
    constructor(sceneManager) {
        this.sceneManager = sceneManager;
        this.lastTime = 0;
        this.running = false;
        this._boundLoop = this._loop.bind(this);
    }

    start() {
        this.running = true;
        this.lastTime = performance.now();
        requestAnimationFrame(this._boundLoop);
    }

    stop() {
        this.running = false;
    }

    _loop(timestamp) {
        if (!this.running) return;

        const dt = Math.min((timestamp - this.lastTime) / 1000, 0.05); // 限制最大dt防止跳帧
        this.lastTime = timestamp;

        this.sceneManager.update(dt);
        this.sceneManager.render();

        requestAnimationFrame(this._boundLoop);
    }
}
```

```javascript
class Renderer {
    constructor(canvas, ctx) {
        this.canvas = canvas;
        this.ctx = ctx;
        this.camera = { x: 0, y: 0 };
        this.screenShake = { intensity: 0, duration: 0, elapsed: 0 };
    }

    clear() {
        this.ctx.clearRect(0, 0, this.canvas.width, this.canvas.height);
    }

    drawImage(img, x, y, w, h, flipX) {
        this.ctx.save();
        if (flipX) {
            this.ctx.translate(x + w, y);
            this.ctx.scale(-1, 1);
            this.ctx.drawImage(img, 0, 0, w, h);
        } else {
            this.ctx.drawImage(img, x, y, w, h);
        }
        this.ctx.restore();
    }

    // 绘制带摄像机偏移的图像
    drawImageWorld(img, worldX, worldY, w, h, flipX) {
        this.drawImage(img, worldX - this.camera.x, worldY - this.camera.y, w, h, flipX);
    }

    drawRect(x, y, w, h, color) {
        this.ctx.fillStyle = color;
        this.ctx.fillRect(x, y, w, h);
    }

    drawRectWorld(x, y, w, h, color) {
        this.drawRect(x - this.camera.x, y - this.camera.y, w, h, color);
    }

    drawText(text, x, y, options = {}) {
        this.ctx.save();
        this.ctx.fillStyle = options.color || '#fff';
        this.ctx.font = options.font || '24px "Courier New", monospace';
        this.ctx.textAlign = options.align || 'left';
        this.ctx.textBaseline = options.baseline || 'top';
        if (options.shadow) {
            this.ctx.shadowColor = options.shadowColor || '#000';
            this.ctx.shadowBlur = options.shadowBlur || 4;
            this.ctx.shadowOffsetX = 2;
            this.ctx.shadowOffsetY = 2;
        }
        this.ctx.fillText(text, x, y);
        this.ctx.restore();
    }

    // 屏幕震动
    shake(intensity, duration) {
        this.screenShake = { intensity, duration, elapsed: 0 };
    }

    updateScreenShake(dt) {
        if (this.screenShake.elapsed < this.screenShake.duration) {
            this.screenShake.elapsed += dt;
            const progress = this.screenShake.elapsed / this.screenShake.duration;
            const currentIntensity = this.screenShake.intensity * (1 - progress);
            this.camera.x += (Math.random() - 0.5) * currentIntensity;
            this.camera.y += (Math.random() - 0.5) * currentIntensity;
        }
    }

    // 淡入淡出覆盖层
    drawOverlay(alpha, color) {
        this.ctx.save();
        this.ctx.fillStyle = color || '#000';
        this.ctx.globalAlpha = alpha;
        this.ctx.fillRect(0, 0, this.canvas.width, this.canvas.height);
        this.ctx.restore();
    }

    setCamera(x, y) {
        this.camera.x = x;
        this.camera.y = y;
    }

    resetCamera() {
        this.camera.x = 0;
        this.camera.y = 0;
    }
}
```

```javascript
class InputHandler {
    constructor() {
        this.keys = {};
        this.keysJustPressed = {};
        this.touches = [];
        this.mousePos = { x: 0, y: 0 };
        this.mouseClicked = false;

        // 键盘事件
        window.addEventListener('keydown', (e) => {
            if (!this.keys[e.code]) {
                this.keysJustPressed[e.code] = true;
            }
            this.keys[e.code] = true;
            // 防止方向键和空格滚动页面
            if (['ArrowUp','ArrowDown','ArrowLeft','ArrowRight','Space'].includes(e.code)) {
                e.preventDefault();
            }
        });

        window.addEventListener('keyup', (e) => {
            this.keys[e.code] = false;
        });

        // 鼠标事件
        window.addEventListener('mousemove', (e) => {
            const canvas = document.getElementById('game-canvas');
            const rect = canvas.getBoundingClientRect();
            const scaleX = CONFIG.CANVAS_WIDTH / rect.width;
            const scaleY = CONFIG.CANVAS_HEIGHT / rect.height;
            this.mousePos.x = (e.clientX - rect.left) * scaleX;
            this.mousePos.y = (e.clientY - rect.top) * scaleY;
        });

        window.addEventListener('click', (e) => {
            this.mouseClicked = true;
        });

        // 触摸事件（移动端适配）
        window.addEventListener('touchstart', (e) => {
            e.preventDefault();
            this.touches = Array.from(e.touches).map(t => {
                const canvas = document.getElementById('game-canvas');
                const rect = canvas.getBoundingClientRect();
                const scaleX = CONFIG.CANVAS_WIDTH / rect.width;
                const scaleY = CONFIG.CANVAS_HEIGHT / rect.height;
                return {
                    x: (t.clientX - rect.left) * scaleX,
                    y: (t.clientY - rect.top) * scaleY
                };
            });
        }, { passive: false });

        window.addEventListener('touchmove', (e) => {
            e.preventDefault();
            this.touches = Array.from(e.touches).map(t => {
                const canvas = document.getElementById('game-canvas');
                const rect = canvas.getBoundingClientRect();
                const scaleX = CONFIG.CANVAS_WIDTH / rect.width;
                const scaleY = CONFIG.CANVAS_HEIGHT / rect.height;
                return {
                    x: (t.clientX - rect.left) * scaleX,
                    y: (t.clientY - rect.top) * scaleY
                };
            });
        }, { passive: false });

        window.addEventListener('touchend', (e) => {
            e.preventDefault();
            this.touches = Array.from(e.touches).map(t => {
                const canvas = document.getElementById('game-canvas');
                const rect = canvas.getBoundingClientRect();
                const scaleX = CONFIG.CANVAS_WIDTH / rect.width;
                const scaleY = CONFIG.CANVAS_HEIGHT / rect.height;
                return {
                    x: (t.clientX - rect.left) * scaleX,
                    y: (t.clientY - rect.top) * scaleY
                };
            });
        }, { passive: false });
    }

    isDown(keyCodes) {
        if (!Array.isArray(keyCodes)) keyCodes = [keyCodes];
        return keyCodes.some(code => this.keys[code]);
    }

    isJustPressed(keyCodes) {
        if (!Array.isArray(keyCodes)) keyCodes = [keyCodes];
        return keyCodes.some(code => this.keysJustPressed[code]);
    }

    isLeft() { return this.isDown(CONFIG.KEYS.LEFT); }
    isRight() { return this.isDown(CONFIG.KEYS.RIGHT); }
    isJump() { return this.isJustPressed(CONFIG.KEYS.UP); }
    isJumpHeld() { return this.isDown(CONFIG.KEYS.UP); }
    isCrouch() { return this.isDown(CONFIG.KEYS.DOWN); }

    // 清除每帧的瞬时状态
    endFrame() {
        this.keysJustPressed = {};
        this.mouseClicked = false;
    }
}
```

```javascript
class CollisionSystem {
    // AABB碰撞检测
    static aabb(a, b) {
        return a.x < b.x + b.w &&
               a.x + a.w > b.x &&
               a.y < b.y + b.h &&
               a.y + a.h > b.y;
    }

    // 圆形碰撞检测
    static circle(a, b) {
        const dx = (a.x + a.r) - (b.x + b.r);
        const dy = (a.y + a.r) - (b.y + b.r);
        const dist = Math.sqrt(dx * dx + dy * dy);
        return dist < (a.r + b.r);
    }

    // AABB vs 圆形
    static aabbCircle(aabb, circle) {
        const closestX = Math.max(aabb.x, Math.min(circle.x + circle.r, aabb.x + aabb.w));
        const closestY = Math.max(aabb.y, Math.min(circle.y + circle.r, aabb.y + aabb.h));
        const dx = (circle.x + circle.r) - closestX;
        const dy = (circle.y + circle.r) - closestY;
        return (dx * dx + dy * dy) < (circle.r * circle.r);
    }

    // 平台碰撞（玩家从上方落下时站在平台上）
    static platformCollision(player, platform) {
        const playerBottom = player.y + player.h;
        const prevPlayerBottom = playerBottom - player.vy * 0.016; // 估算上一帧位置

        return player.x + player.w > platform.x &&
               player.x < platform.x + platform.w &&
               playerBottom >= platform.y &&
               prevPlayerBottom <= platform.y + 10 && // 允许10px容差
               player.vy >= 0; // 只在落下时检测
    }
}
```

```javascript
class AudioManager {
    constructor() {
        this.sounds = {};
        this.bgm = null;
        this.bgmVolume = 0.5;
        this.sfxVolume = 0.7;
        this.muted = false;

        // 从localStorage读取音量设置
        const saved = localStorage.getItem(CONFIG.SAVE_KEYS.VOLUME);
        if (saved) {
            const vol = JSON.parse(saved);
            this.bgmVolume = vol.bgm || 0.5;
            this.sfxVolume = vol.sfx || 0.7;
        }
    }

    loadSound(name, path) {
        const audio = new Audio(path);
        audio.preload = 'auto';
        this.sounds[name] = audio;
    }

    playSFX(name, volume) {
        if (this.muted) return;
        const sound = this.sounds[name];
        if (!sound) return;
        sound.volume = volume || this.sfxVolume;
        sound.currentTime = 0;
        sound.play().catch(() => {}); // 忽略自动播放策略错误
    }

    playBGM(name, loop) {
        if (this.muted) return;
        this.stopBGM();
        const sound = this.sounds[name];
        if (!sound) return;
        sound.loop = loop !== false;
        sound.volume = this.bgmVolume;
        sound.currentTime = 0;
        sound.play().catch(() => {});
        this.bgm = sound;
    }

    stopBGM() {
        if (this.bgm) {
            this.bgm.pause();
            this.bgm = null;
        }
    }

    setVolume(bgm, sfx) {
        if (bgm !== undefined) this.bgmVolume = bgm;
        if (sfx !== undefined) this.sfxVolume = sfx;
        localStorage.setItem(CONFIG.SAVE_KEYS.VOLUME, JSON.stringify({
            bgm: this.bgmVolume,
            sfx: this.sfxVolume
        }));
    }

    setMuted(muted) {
        this.muted = muted;
        if (muted) this.stopBGM();
    }
}
```

```javascript
class ParticleSystem {
    constructor() {
        this.particles = [];
    }

    // 添加粒子
    emit(x, y, options = {}) {
        const count = options.count || 10;
        for (let i = 0; i < count; i++) {
            this.particles.push({
                x: x,
                y: y,
                vx: (Math.random() - 0.5) * (options.spreadX || 200),
                vy: (Math.random() - 0.5) * (options.spreadY || 200) - (options.upwardBias || 0),
                life: options.life || 1.0,
                maxLife: options.life || 1.0,
                size: options.size || (Math.random() * 4 + 2),
                color: options.colors ? options.colors[Math.floor(Math.random() * options.colors.length)] : '#fff',
                gravity: options.gravity || 0,
                shape: options.shape || 'circle', // 'circle', 'rect', 'star'
                rotation: Math.random() * Math.PI * 2,
                rotSpeed: (Math.random() - 0.5) * 5
            });
        }
    }

    // 跳跃尘土
    emitDust(x, y) {
        this.emit(x, y, {
            count: 8,
            spreadX: 100,
            spreadY: 50,
            life: 0.4,
            size: 3,
            colors: ['#aaa', '#888', '#666'],
            gravity: 200,
            shape: 'circle'
        });
    }

    // 收集道具星光
    emitCollect(x, y) {
        this.emit(x, y, {
            count: 20,
            spreadX: 150,
            spreadY: 150,
            life: 0.6,
            size: 4,
            colors: ['#ff0', '#fff', '#0ff', '#f0f'],
            shape: 'star',
            upwardBias: 100
        });
    }

    // 镭射击中爆炸
    emitExplosion(x, y) {
        this.emit(x, y, {
            count: 30,
            spreadX: 200,
            spreadY: 200,
            life: 0.5,
            size: 5,
            colors: ['#f00', '#f80', '#ff0', '#fff'],
            shape: 'rect',
            gravity: 100
        });
    }

    // 道具获得光环弹跳
    emitPropGet(x, y, color) {
        this.emit(x, y, {
            count: 15,
            spreadX: 80,
            spreadY: 80,
            life: 0.8,
            size: 6,
            colors: [color, '#fff'],
            shape: 'circle',
            upwardBias: 150
        });
    }

    // 速度线效果
    emitSpeedLines(y) {
        for (let i = 0; i < 5; i++) {
            this.particles.push({
                x: CONFIG.CANVAS_WIDTH + 50,
                y: Math.random() * CONFIG.CANVAS_HEIGHT,
                vx: -800 - Math.random() * 400,
                vy: 0,
                life: 0.3,
                maxLife: 0.3,
                size: 2,
                color: 'rgba(255,255,255,0.3)',
                shape: 'rect',
                isLine: true,
                lineWidth: 40 + Math.random() * 60
            });
        }
    }

    // 数据化溶解效果
    emitDataDissolve(x, y, w, h) {
        for (let i = 0; i < 40; i++) {
            this.particles.push({
                x: x + Math.random() * w,
                y: y + Math.random() * h,
                vx: (Math.random() - 0.5) * 100,
                vy: -50 - Math.random() * 100,
                life: 0.6,
                maxLife: 0.6,
                size: 4 + Math.random() * 4,
                color: `hsl(${200 + Math.random() * 40}, 100%, ${50 + Math.random() * 30}%)`,
                shape: 'rect',
                gravity: -50
            });
        }
    }

    update(dt) {
        for (let i = this.particles.length - 1; i >= 0; i--) {
            const p = this.particles[i];
            p.x += p.vx * dt;
            p.y += p.vy * dt;
            p.vy += p.gravity * dt;
            p.life -= dt;
            p.rotation += p.rotSpeed * dt;
            if (p.life <= 0) {
                this.particles.splice(i, 1);
            }
        }
    }

    render(ctx) {
        for (const p of this.particles) {
            const alpha = Math.max(0, p.life / p.maxLife);
            ctx.save();
            ctx.globalAlpha = alpha;
            ctx.translate(p.x, p.y);
            ctx.rotate(p.rotation);

            if (p.isLine) {
                ctx.fillStyle = p.color;
                ctx.fillRect(-p.lineWidth / 2, -p.size / 2, p.lineWidth, p.size);
            } else if (p.shape === 'circle') {
                ctx.fillStyle = p.color;
                ctx.beginPath();
                ctx.arc(0, 0, p.size, 0, Math.PI * 2);
                ctx.fill();
            } else if (p.shape === 'rect') {
                ctx.fillStyle = p.color;
                ctx.fillRect(-p.size / 2, -p.size / 2, p.size, p.size);
            } else if (p.shape === 'star') {
                ctx.fillStyle = p.color;
                this._drawStar(ctx, 0, 0, 5, p.size, p.size / 2);
            }

            ctx.restore();
        }
    }

    _drawStar(ctx, cx, cy, spikes, outerR, innerR) {
        let rot = Math.PI / 2 * 3;
        const step = Math.PI / spikes;
        ctx.beginPath();
        ctx.moveTo(cx, cy - outerR);
        for (let i = 0; i < spikes; i++) {
            ctx.lineTo(cx + Math.cos(rot) * outerR, cy + Math.sin(rot) * outerR);
            rot += step;
            ctx.lineTo(cx + Math.cos(rot) * innerR, cy + Math.sin(rot) * innerR);
            rot += step;
        }
        ctx.closePath();
        ctx.fill();
    }

    clear() {
        this.particles = [];
    }
}
```

```javascript
class SceneManager {
    constructor(renderer, input, audio, collision, particles, uiLayer) {
        this.renderer = renderer;
        this.input = input;
        this.audio = audio;
        this.collision = collision;
        this.particles = particles;
        this.uiLayer = uiLayer;
        this.currentScene = null;
        this.transitionAlpha = 0;
        this.transitioning = false;
        this.nextScene = null;
        this.sceneData = {};
    }

    changeScene(sceneName, data) {
        if (this.transitioning) return;
        this.nextScene = sceneName;
        this.sceneData = data || {};
        this.transitioning = true;
        this.transitionAlpha = 0;
        this.audio.playSFX('sfx_transition');
    }

    update(dt) {
        // 场景过渡处理
        if (this.transitioning) {
            this.transitionAlpha += dt * 2.5; // 0.4秒过渡
            if (this.transitionAlpha >= 1) {
                this.transitionAlpha = 1;
                // 切换场景
                this._loadScene(this.nextScene);
                // 开始淡出
                setTimeout(() => {
                    this._fadeOut = true;
                }, 100);
            }
            return;
        }

        if (this._fadeOut) {
            this.transitionAlpha -= dt * 2.5;
            if (this.transitionAlpha <= 0) {
                this.transitionAlpha = 0;
                this._fadeOut = false;
            }
        }

        if (this.currentScene) {
            this.currentScene.update(dt);
        }
    }

    render() {
        this.renderer.clear();

        if (this.currentScene) {
            this.currentScene.render();
        }

        // 粒子渲染（在世界空间）
        this.particles.render(this.renderer.ctx);

        // 过渡覆盖层
        if (this.transitionAlpha > 0) {
            this.renderer.drawOverlay(this.transitionAlpha);
        }
    }

    _loadScene(name) {
        this.uiLayer.innerHTML = ''; // 清除UI层
        this.particles.clear();
        this.renderer.resetCamera();

        const params = {
            renderer: this.renderer,
            input: this.input,
            audio: this.audio,
            collision: this.collision,
            particles: this.particles,
            uiLayer: this.uiLayer,
            data: this.sceneData,
            changeScene: (s, d) => this.changeScene(s, d)
        };

        switch (name) {
            case 'boot':
                this.currentScene = new BootScene(params);
                break;
            case 'mainMenu':
                this.currentScene = new MainMenuScene(params);
                break;
            case 'characterSelect':
                this.currentScene = new CharacterSelectScene(params);
                break;
            case 'levelSelect':
                this.currentScene = new LevelSelectScene(params);
                break;
            case 'tencentLevel':
                this.currentScene = new TencentLevel(params);
                break;
            case 'xpengStage1':
                this.currentScene = new XpengStage1(params);
                break;
            case 'xpengStage2':
                this.currentScene = new XpengStage2(params);
                break;
            case 'xpengEnding':
                this.currentScene = new XpengEnding(params);
                break;
            case 'gameOver':
                this.currentScene = new GameOverScene(params);
                break;
            default:
                console.warn('Unknown scene:', name);
        }
    }
}
```

```javascript
class Player extends Entity {
    constructor(x, y, gender) {
        super(x, y, 48, 72);
        this.gender = gender; // 'male' or 'female'
        this.onGround = false;
        this.crouching = false;
        this.jumpCount = 0;
        this.maxJumps = 1; // 默认一跳，关卡2可设为2
        this.invincibleTimer = 0;
        this.facingRight = true;
        this.animFrame = 0;
        this.animTimer = 0;
        this.animSpeed = 0.1; // 每帧秒数

        // 动画精灵图（待加载后赋值）
        this.sprites = {
            idle: null,
            run: null,
            jump: null,
            crouch: null,
            death: null
        };
        this.frameCount = { idle: 4, run: 8, jump: 2, crouch: 2, death: 4 };
    }

    update(dt, input, platforms) {
        // 水平移动
        const speed = this.crouching ? CONFIG.PLAYER_CROUCH_SPEED : CONFIG.PLAYER_SPEED;
        if (input.isLeft()) {
            this.vx = -speed;
            this.facingRight = false;
        } else if (input.isRight()) {
            this.vx = speed;
            this.facingRight = true;
        } else {
            this.vx *= 0.8; // 摩擦力减速
        }

        // 下蹲
        this.crouching = input.isCrouch() && this.onGround;
        if (this.crouching) {
            this.h = 36; // 下蹲高度减半
        } else {
            this.h = 72;
        }

        // 跳跃
        if (input.isJump()) {
            if (this.onGround) {
                this.vy = CONFIG.JUMP_FORCE;
                this.onGround = false;
                this.jumpCount = 1;
                this.invincibleTimer = CONFIG.COLLISION.INVINCIBLE_FRAMES;
            } else if (this.jumpCount < this.maxJumps) {
                this.vy = CONFIG.DOUBLE_JUMP_FORCE;
                this.jumpCount++;
                this.invincibleTimer = CONFIG.COLLISION.INVINCIBLE_FRAMES;
            }
        }

        // 重力
        this.vy += CONFIG.GRAVITY * dt;

        // 更新位置
        super.update(dt);

        // 平台碰撞
        this.onGround = false;
        for (const platform of platforms) {
            if (CollisionSystem.platformCollision(this, platform)) {
                this.y = platform.y - this.h;
                this.vy = 0;
                this.onGround = true;
                this.jumpCount = 0;
            }
        }

        // 无敌帧倒计时
        if (this.invincibleTimer > 0) {
            this.invincibleTimer -= dt;
        }

        // 动画帧更新
        this.animTimer += dt;
        if (this.animTimer >= this.animSpeed) {
            this.animTimer = 0;
            this.animFrame = (this.animFrame + 1) % this.frameCount.run;
        }
    }

    render(renderer) {
        // 无敌帧闪烁
        if (this.invincibleTimer > 0 && Math.floor(this.invincibleTimer * 20) % 2 === 0) {
            return;
        }

        // 确定当前动画
        let spriteKey = 'idle';
        let frameIdx = 0;

        if (!this.onGround) {
            spriteKey = 'jump';
            frameIdx = this.jumpCount >= 2 ? 1 : 0;
        } else if (this.crouching) {
            spriteKey = 'crouch';
            frameIdx = 0;
        } else if (Math.abs(this.vx) > 10) {
            spriteKey = 'run';
            frameIdx = this.animFrame % this.frameCount.run;
        } else {
            spriteKey = 'idle';
            frameIdx = Math.floor(Date.now() / 250) % this.frameCount.idle;
        }

        const sprite = this.sprites[spriteKey];
        if (sprite) {
            const frameW = sprite.width / this.frameCount[spriteKey];
            const frameH = sprite.height;
            const sx = frameIdx * frameW;

            renderer.drawImage(
                sprite,
                this.x - renderer.camera.x,
                this.y - renderer.camera.y,
                this.w,
                this.h,
                !this.facingRight,
                sx, 0, frameW, frameH
            );
        } else {
            // 无贴图时绘制占位矩形
            renderer.drawRectWorld(this.x, this.y, this.w, this.h, '#4af');
        }
    }

    getHitbox() {
        // 下蹲时碰撞盒高度减半
        const h = this.crouching ? this.h : this.h;
        return { x: this.x + 8, y: this.y + 4, w: this.w - 16, h: h - 8 };
    }
}
```

```javascript
class DialogBox {
    constructor(uiLayer) {
        this.uiLayer = uiLayer;
        this.element = null;
    }

    show(text, options = {}) {
        this.hide();
        const div = document.createElement('div');
        div.className = 'dialog-box';
        div.style.left = options.x ? options.x + 'px' : '50%';
        div.style.top = options.y ? options.y + 'px' : '50%';
        if (!options.x) div.style.transform = 'translate(-50%, -50%)';
        div.innerHTML = text;
        if (options.onClick) {
            div.style.cursor = 'pointer';
            div.addEventListener('click', options.onClick);
        }
        this.uiLayer.appendChild(div);
        this.element = div;
    }

    hide() {
        if (this.element) {
            this.element.remove();
            this.element = null;
        }
    }
}

// js/ui/Toast.js
class Toast {
    constructor(uiLayer) {
        this.uiLayer = uiLayer;
        this.element = null;
        this.timer = null;
    }

    show(text, duration) {
        this.hide();
        const div = document.createElement('div');
        div.className = 'toast';
        div.textContent = text;
        this.uiLayer.appendChild(div);
        this.element = div;

        requestAnimationFrame(() => div.classList.add('show'));

        this.timer = setTimeout(() => this.hide(), duration || 2000);
    }

    hide() {
        if (this.timer) clearTimeout(this.timer);
        if (this.element) {
            this.element.classList.remove('show');
            setTimeout(() => {
                if (this.element) this.element.remove();
                this.element = null;
            }, 300);
        }
    }
}
```

```javascript
class HUD {
    constructor(uiLayer, audio) {

    initPropSlots() {
        const slotSize = 56;
        const gap = 8;
        const totalW = 6 * slotSize + 5 * gap;
        const startX = CONFIG.CANVAS_WIDTH - totalW - 30;
        const startY = 30;

        this.propSlots = [];
        const propNames = ['微信绿气泡', 'QQ企鹅晶石', '游戏手柄', '云朵芯片', '视频播放键', '扳手齿轮'];
        const propColors = ['#07c160', '#12b7f5', '#ffd700', '#e8f4ff', '#ff4444', '#c0c0c0'];

        for (let i = 0; i < 6; i++) {
            this.propSlots.push({
                x: startX + i * (slotSize + gap),
                y: startY,
                w: slotSize,
                h: slotSize,
                filled: false,
                name: propNames[i],
                color: propColors[i],
                bounceTimer: 0
            });
        }
    }

    // 填充道具槽
    fillPropSlot(index) {
        if (index >= 0 && index < this.propSlots.length) {
            this.propSlots[index].filled = true;
            this.propSlots[index].bounceTimer = 0.5;
            this.audio.playSFX('sfx_collect_prop');
        }
    }

    // 初始化体力条（小鹏关卡用）
    initStaminaBar() {
        this.staminaBar = {
            x: 30,
            y: 30,
            w: 300,
            h: 24,
            value: 100,
            maxValue: 100
        };
    }

    updateStamina(value) {
        if (this.staminaBar) {
            this.staminaBar.value = Math.max(0, Math.min(100, value));
        }
    }

    showMessage(text, duration) {
        this.messageText = text;
        this.messageTimer = duration || 2;
    }

    update(dt) {
        // 道具弹跳动画
        for (const slot of this.propSlots) {
            if (slot.bounceTimer > 0) {
                slot.bounceTimer -= dt;
            }
        }
        if (this.messageTimer > 0) {
            this.messageTimer -= dt;
        }
    }

    render(ctx) {
        // 道具栏
        for (const slot of this.propSlots) {
            let bounceY = 0;
            if (slot.bounceTimer > 0) {
                bounceY = Math.sin(slot.bounceTimer * Math.PI * 4) * 8;
            }

            ctx.save();
            if (slot.filled) {
                ctx.fillStyle = slot.color;
                ctx.globalAlpha = 0.3;
                ctx.fillRect(slot.x - 2, slot.y - 2 + bounceY, slot.w + 4, slot.h + 4);
                ctx.globalAlpha = 1;
                ctx.strokeStyle = slot.color;
                ctx.lineWidth = 2;
            } else {
                ctx.strokeStyle = '#555';
                ctx.lineWidth = 1;
            }
            ctx.beginPath();
            ctx.roundRect(slot.x, slot.y + bounceY, slot.w, slot.h, 8);
            ctx.stroke();

            if (slot.filled) {
                ctx.fillStyle = '#fff';
                ctx.font = '10px "Courier New", monospace';
                ctx.textAlign = 'center';
                ctx.fillText(slot.name, slot.x + slot.w / 2, slot.y + slot.h + 14 + bounceY);
            }
            ctx.restore();
        }

        // 体力条
        if (this.staminaBar) {
            const bar = this.staminaBar;
            const ratio = bar.value / bar.maxValue;

            // 背景
            ctx.fillStyle = 'rgba(0,0,0,0.6)';
            ctx.beginPath();
            ctx.roundRect(bar.x, bar.y, bar.w, bar.h, 6);
            ctx.fill();

            // 填充颜色
            let fillColor = '#0f0';
            if (ratio < 0.1) fillColor = '#f00';
            else if (ratio < 0.3) fillColor = '#f80';
            else if (ratio < 0.6) fillColor = '#ff0';

            ctx.fillStyle = fillColor;
            ctx.beginPath();
            ctx.roundRect(bar.x + 2, bar.y + 2, (bar.w - 4) * ratio, bar.h - 4, 4);
            ctx.fill();

            // 文字
            ctx.fillStyle = '#fff';
            ctx.font = '14px "Courier New", monospace';
            ctx.textAlign = 'left';
            ctx.fillText(`体力: ${Math.ceil(bar.value)}%`, bar.x + 8, bar.y + 17);

            // 低体力闪烁
            if (ratio < 0.1 && Math.floor(Date.now() / 200) % 2 === 0) {

                ctx.lineWidth = 2;
                ctx.beginPath();
                ctx.roundRect(bar.x - 2, bar.y - 2, bar.w + 4, bar.h + 4, 8);
                ctx.stroke();
            }
        }

        // 消息提示
        if (this.messageTimer > 0) {

            ctx.globalAlpha = Math.min(1, this.messageTimer);
            ctx.fillStyle = 'rgba(0,0,0,0.7)';

            ctx.textAlign = 'center';

            ctx.restore();
        }
    }
```

```javascript
class TencentPuzzles {
    constructor(renderer, input, audio, particles) {
        this.renderer = renderer;
        this.input = input;
        this.audio = audio;
        this.particles = particles;
        this.currentPuzzle = null;
        this.solved = false;
    }

    // 启动解密
    startPuzzle(type) {
        this.currentPuzzle = type;
        this.solved = false;
        this.audio.playBGM('bgm_tencent_puzzle');

        switch (type) {
            case 'wechat':
                this._initWechatPuzzle();
                break;
            case 'qq':
                this._initQQPuzzle();
                break;
            case 'games':
                this._initGamesPuzzle();
                break;
            case 'cloud':
                this._initCloudPuzzle();
                break;
            case 'content':
                this._initContentPuzzle();
                break;
            case 'tech':
                this._initTechPuzzle();
                break;
        }
    }

    _initWechatPuzzle() {
        // 滑动拼图 - 拼出微信小程序图标
        this.puzzleState = {
            type: 'slide',
            gridSize: 3,
            tiles: [],
            selectedTile: null
        };
        // 初始化9宫格（最后一格为空）
        for (let i = 0; i < 9; i++) {
            this.puzzleState.tiles.push({ id: i, currentPos: i });
        }
        // 打乱
        this._shuffleTiles();
    }

    _initQQPuzzle() {
        // 顺序踩按钮 - 童年→校园→工作→未来
        this.puzzleState = {
            type: 'sequence',
            buttons: ['童年', '校园', '工作', '未来'],
            correctOrder: [0, 1, 2, 3],
            playerSequence: [],
            phase: 'input' // 'input' or 'result'
        };
    }

    _initGamesPuzzle() {
        // 击败3波AI敌人
        this.puzzleState = {
            type: 'combat',
            wave: 1,
            maxWaves: 3,
            enemies: [],
            playerSafe: true
        };
        this._spawnWave();
    }

    _initCloudPuzzle() {
        // 3×3灯阵记忆游戏
        this.puzzleState = {
            type: 'memory',
            gridSize: 3,
            lights: Array(9).fill(false),
            sequence: [],
            playerSequence: [],
            phase: 'showing', // 'showing' or 'input'
            showIndex: 0,
            showTimer: 0
        };
        // 生成随机序列
        for (let i = 0; i < 5; i++) {
            this.puzzleState.sequence.push(Math.floor(Math.random() * 9));
        }
    }

    _initContentPuzzle() {
        // 视频片段排序 - 拍摄→剪辑→特效→发布
        this.puzzleState = {
            type: 'sort',
            clips: ['拍摄', '剪辑', '特效', '发布'],
            shuffled: [],
            selected: [],
            correct: [0, 1, 2, 3]
        };
        // 打乱
        this.puzzleState.shuffled = [...this.puzzleState.clips].sort(() => Math.random() - 0.5);
    }

    _initTechPuzzle() {
        // 密码输入 - 1024
        this.puzzleState = {
            type: 'code',
            code: '1024',
            input: '',
            hints: ['墙上公式: E=mc²', '便签: 2的10次方', '摩斯: ·— —·· ·—·']
        };
    }

    _shuffleTiles() {
        // Fisher-Yates 洗牌
        const tiles = this.puzzleState.tiles;
        for (let i = tiles.length - 1; i > 0; i--) {
            const j = Math.floor(Math.random() * (i + 1));
            [tiles[i].currentPos, tiles[j].currentPos] = [tiles[j].currentPos, tiles[i].currentPos];
        }
    }

    _spawnWave() {
        this.puzzleState.enemies = [];
        const count = this.puzzleState.wave + 2;
        for (let i = 0; i < count; i++) {
            this.puzzleState.enemies.push({
                x: 400 + Math.random() * 800,
                y: 300 + Math.random() * 300,
                w: 48,
                h: 48,
                alive: true,
                vx: (Math.random() - 0.5) * 100,
                vy: (Math.random() - 0.5) * 100
            });
        }
    }

    update(dt) {
        if (!this.currentPuzzle || this.solved) return;

        switch (this.puzzleState.type) {
            case 'memory':
                this._updateMemoryPuzzle(dt);
                break;
            case 'combat':
                this._updateCombatPuzzle(dt);
                break;
        }
    }

    _updateMemoryPuzzle(dt) {
        const ps = this.puzzleState;
        if (ps.phase === 'showing') {
            ps.showTimer += dt;
            if (ps.showTimer > 0.8) {
                ps.showTimer = 0;
                // 熄灭上一个
                if (ps.showIndex > 0) {
                    ps.lights[ps.sequence[ps.showIndex - 1]] = false;
                }
                if (ps.showIndex < ps.sequence.length) {
                    ps.lights[ps.sequence[ps.showIndex]] = true;
                    ps.showIndex++;
                } else {
                    // 演示完毕，全部熄灭，进入玩家输入
                    ps.lights = Array(9).fill(false);
                    ps.phase = 'input';
                    ps.playerSequence = [];
                }
            }
        }
    }

    _updateCombatPuzzle(dt) {
        for (const enemy of this.puzzleState.enemies) {
            if (!enemy.alive) continue;
            enemy.x += enemy.vx * dt;
            enemy.y += enemy.vy * dt;
            // 边界反弹
            if (enemy.x < 200 || enemy.x > 1400) enemy.vx *= -1;
            if (enemy.y < 200 || enemy.y > 700) enemy.vy *= -1;
        }
    }

    // 处理玩家点击解密界面
    handleClick(x, y) {
        if (!this.currentPuzzle || this.solved) return false;

        switch (this.puzzleState.type) {
            case 'slide':
                return this._handleSlideClick(x, y);
            case 'sequence':
                return this._handleSequenceClick(x, y);
            case 'memory':
                return this._handleMemoryClick(x, y);
            case 'sort':
                return this._handleSortClick(x, y);
            case 'code':
                return this._handleCodeClick(x, y);
            case 'combat':
                return this._handleCombatClick(x, y);
        }
        return false;
    }

    _handleSlideClick(x, y) {
        // 简化：点击任意未归位方块视为正确移动
        const ps = this.puzzleState;
        const allCorrect = ps.tiles.every((t, i) => t.currentPos === i);
        if (allCorrect) {
            this._puzzleSolved();
            return true;
        }
        // 随机移动一个方块（演示用）
        this._shuffleTiles();
        return true;
    }

    _handleSequenceClick(x, y) {
        const ps = this.puzzleState;
        // 简化：按顺序点击即正确
        ps.playerSequence.push(ps.playerSequence.length);
        if (ps.playerSequence.length === ps.correctOrder.length) {
            this._puzzleSolved();
            return true;
        }
        return true;
    }

    _handleMemoryClick(x, y) {
        const ps = this.puzzleState;
        if (ps.phase !== 'input') return false;

        // 计算点击了哪个灯
        const gridX = Math.floor((x - 300) / 120);
        const gridY = Math.floor((y - 200) / 120);
        const idx = gridY * 3 + gridX;

        if (idx >= 0 && idx < 9) {
            ps.playerSequence.push(idx);
            ps.lights[idx] = true;

            // 检查当前步骤是否正确
            const step = ps.playerSequence.length - 1;
            if (ps.playerSequence[step] !== ps.sequence[step]) {
                // 错误，重置
                this.audio.playSFX('sfx_puzzle_wrong');
```

```javascript
        // 速度递增

        const targetSpeed = CONFIG.XPENG.STAGE1_BASE_SPEED * (1 + speedUpCount * CONFIG.XPENG.STAGE1_SPEED_INCREMENT);
        const clampedSpeed = Math.min(targetSpeed, CONFIG.XPENG.STAGE1_MAX_SPEED);

            this.scrollSpeed = clampedSpeed;
            this.audio.playSFX('sfx_speed_up');
            this.particles.emitSpeedLines(0);

                this.audio.playSFX('sfx_jump');

            }

        const targetY = this._laneY(this.currentLane);
        this.player.y += (targetY - this.player.y) * 8 * dt;

        // 体力消耗
        let staminaDrain = CONFIG.XPENG.STAMINA_DRAIN_RATE;
        if (this.player.crouching) staminaDrain += CONFIG.XPENG.STAMINA_CROUCH_RATE;
        const currentStamina = this.hud.staminaBar.value - staminaDrain * dt;
        this.hud.updateStamina(currentStamina);

        // 体力低音效
        if (currentStamina < 10 && Math.floor(Date.now() / 500) % 2 === 0) {
            this.audio.playSFX('sfx_stamina_low');

            obs.update(dt);
```

```javascript
class XpengStage2 {
    constructor(params) {

    _laneY(lane) {

    }

    update(dt) {
        this.elapsedTime += dt;

        // 跑道切换
        if (this.input.isJustPressed(CONFIG.KEYS.UP)) {
            if (this.currentLane > 0) {
                this.currentLane--;
                this.audio.playSFX('sfx_jump');
            }
        }
        if (this.input.isJustPressed(CONFIG.KEYS.DOWN)) {
            if (this.currentLane < this.laneCount - 1) {
                this.currentLane++;
                this.audio.playSFX('sfx_crouch');
            }
        }

        // 平滑切换
        const targetY = this._laneY(this.currentLane);
        this.player.y += (targetY - this.player.y) * 10 * dt;

        // 难度递增 - 根据时间生成不同障碍
        this.obstacleTimer += dt;
        const interval = this._getObstacleInterval();
        if (this.obstacleTimer >= interval) {
            this.obstacleTimer = 0;
            this._spawnObstacleByPhase();
        }

        // 更新障碍
        for (let i = this.obstacles.length - 1; i >= 0; i--) {
            const obs = this.obstacles[i];
            obs.x -= this.scrollSpeed * dt;
            obs.update(dt);

            // 一击必杀
            if (CollisionSystem.aabb(this.player.getHitbox(), obs.getBounds())) {
                this._playerDeath('被镭射击中！');
                return;
            }

            if (obs.x + obs.w < -50) {
                this.obstacles.splice(i, 1);
            }
        }

        // 镭射预警音效
        for (const obs of this.obstacles) {
            if (obs.type.includes('laser') && obs.warningTimer > 0 && obs.warningTimer < dt * 2) {
                this.audio.playSFX('sfx_laser_warning');
            }
        }

        // 阶段二结束
        if (this.elapsedTime >= CONFIG.XPENG.STAGE2_DURATION) {

                gender: this.gender,
                stage1Time: this.stage1Time,
                stage2Time: this.elapsedTime,
                batteriesCollected: this.batteriesCollected,
                deaths: this.deathCount
            });

    _getObstacleInterval() {
        if (this.elapsedTime < 30) return 2.0;
        if (this.elapsedTime < 60) return 1.5;
        if (this.elapsedTime < 90) return 1.2;
        if (this.elapsedTime < 120) return 1.0;
        return 0.75; // 120秒后频率提升25%

    _spawnObstacleByPhase() {
        const lane = Math.floor(Math.random() * this.laneCount);
        const y = this._laneY(lane);

        // 确保至少有一条跑道是安全的
        const occupiedLanes = new Set();
        for (const obs of this.obstacles) {
            if (obs.x > CONFIG.CANVAS_WIDTH - 300) {
                // 找到这个障碍所在的跑道
                for (let l = 0; l < this.laneCount; l++) {
                    if (Math.abs(obs.y - this._laneY(l)) < this.laneHeight / 2) {
                        occupiedLanes.add(l);
                    }
                }
            }
        }

        // 如果只剩一条跑道，不再生成障碍
        if (occupiedLanes.size >= this.laneCount - 1) return;

        // 选择未被占用的跑道
        let targetLane = lane;
        while (occupiedLanes.has(targetLane)) {
            targetLane = (targetLane + 1) % this.laneCount;
        }
        const safeY = this._laneY(targetLane);

        let type, w, h;
        if (this.elapsedTime < 30) {
            type = 'laser_ground'; w = 16; h = 200;
        } else if (this.elapsedTime < 60) {
            type = Math.random() > 0.5 ? 'laser_ground' : 'laser_air';
            w = type === 'laser_air' ? 200 : 16;
            h = type === 'laser_air' ? 16 : 200;
        } else if (this.elapsedTime < 90) {
            const types = ['laser_ground', 'laser_air', 'laser_cross'];
            type = types[Math.floor(Math.random() * types.length)];
            w = type === 'laser_cross' ? 160 : (type === 'laser_air' ? 200 : 16);
            h = type === 'laser_cross' ? 160 : (type === 'laser_air' ? 16 : 200);
        } else if (this.elapsedTime < 120) {
            const types = ['laser_ground', 'laser_air', 'laser_cross', 'laser_tracking'];
            type = types[Math.floor(Math.random() * types.length)];
            w = type === 'laser_tracking' ? 16 : (type === 'laser_cross' ? 160 : (type === 'laser_air' ? 200 : 16));
            h = type === 'laser_tracking' ? 200 : (type === 'laser_cross' ? 160 : (type === 'laser_air' ? 16 : 200));
        } else {
            const types = ['laser_ground', 'laser_air', 'laser_cross', 'laser_tracking', 'laser_rotating', 'cable'];
            type = types[Math.floor(Math.random() * types.length)];
            w = 64; h = 64;
            if (type === 'laser_rotating') { w = 96; h = 96; }
            if (type === 'cable') { w = 32; h = 128; }
        }

        const obs = new Obstacle(CONFIG.CANVAS_WIDTH + 50, safeY - h / 2, w, h, type);
        obs.warningTimer = this.elapsedTime < 60 ? 2.0 : (this.elapsedTime < 120 ? 1.2 : 1.0);
        this.obstacles.push(obs);
    }

    _playerDeath(reason) {
        this.deathCount++;
        this.renderer.shake(15, 0.5);
        this.audio.playSFX('sfx_laser_hit');
        this.particles.emitExplosion(this.player.x + this.player.w / 2, this.player.y + this.player.h / 2);

        this.changeScene('gameOver', {
            level: 'xpeng_stage2',
            reason: reason,
            stage1Time: this.stage1Time,
            stage2Time: this.elapsedTime,
            deaths: this.deathCount,
            batteriesCollected: this.batteriesCollected
        });
    }

    render() {
        const ctx = this.renderer.ctx;

        // 昏暗背景
        ctx.fillStyle = '#050a15';
        ctx.fillRect(0, 0, CONFIG.CANVAS_WIDTH, CONFIG.CANVAS_HEIGHT);

        // 跑道线（更密集）
        for (let i = 0; i <= this.laneCount; i++) {

            ctx.lineWidth = 1;
            ctx.beginPath();
            ctx.moveTo(0, y);
            ctx.lineTo(CONFIG.CANVAS_WIDTH, y);
            ctx.stroke();
        }

        // 障碍（镭射发光效果）

                ctx.shadowColor = '#f00';
                ctx.shadowBlur = 15;
            }
            obs.render(this.renderer);
            ctx.restore();
        }

        // 玩家

        // 计时
        ctx.fillStyle = '#f44';
        ctx.font = '20px "Courier New"';
```

```javascript
        this.phaseTimer = 0;
        this.weatherPhase = 0; // 0: 正常, 1: 暴雨, 2: 雷电, 3: 浓雾, 4: 结冰, 5: 晴朗

        // 汽车位置

        // 播放BGM
        this.audio.playBGM('bgm_xpeng_ending', false);
        this.audio.playSFX('sfx_car_engine_start');
    }

    update(dt) {
        this.phaseTimer += dt;

        switch (this.phase) {
            case 0: // 上车动画（2秒）
                this.carX += (this.carTargetX - this.carX) * 2 * dt;
                if (this.phaseTimer > 2) {
                    this.phase = 1;
                    this.phaseTimer = 0;
                }
                break;
            case 1: // 汽车启动（1.5秒）
                if (this.phaseTimer > 1.5) {
                    this.phase = 2;
                    this.phaseTimer = 0;
                    this.audio.playSFX('sfx_car_drive');
                }
                break;
            case 2: // 行驶展示（8秒，包含天气变化）
                // 天气切换
                const weatherDuration = 1.5;
                this.weatherPhase = Math.floor(this.phaseTimer / weatherDuration);

                if (this.phaseTimer > 8) {
                    this.phase = 3;
                    this.phaseTimer = 0;
                }
                break;
            case 3: // 文字展示（3秒）
                if (this.phaseTimer > 3) {
                    this.phase = 4;
                    this.phaseTimer = 0;
                }
                break;
            case 4: // 统计 + 按钮
                if (this.input.mouseClicked) {
                    this.changeScene('gameOver', {
                        level: 'xpeng_complete',
                        stage1Time: this.stage1Time,
                        stage2Time: this.stage2Time,
                        batteriesCollected: this.batteriesCollected,
                        deaths: this.deathCount
                    });
                }
                break;
        }
    }

    render() {
        const ctx = this.renderer.ctx;

        // 背景渐变
        if (this.phase >= 3 && this.weatherPhase >= 4) {
            // 阳光明媚

            ctx.fillStyle = '#0a1525';
        }
        ctx.fillRect(0, 0, CONFIG.CANVAS_WIDTH, CONFIG.CANVAS_HEIGHT);

        // 道路
        ctx.fillStyle = '#333';
        ctx.fillRect(0, CONFIG.CANVAS_HEIGHT - 200, CONFIG.CANVAS_WIDTH, 200);
        // 道路标线
        ctx.strokeStyle = '#ff0';
        ctx.lineWidth = 3;
        ctx.setLineDash([40, 20]);
        ctx.beginPath();
        ctx.moveTo(0, CONFIG.CANVAS_HEIGHT - 100);
        ctx.lineTo(CONFIG.CANVAS_WIDTH, CONFIG.CANVAS_HEIGHT - 100);
        ctx.stroke();
        ctx.setLineDash([]);

        // 汽车
        ctx.fillStyle = '#4af';
        ctx.fillRect(this.carX, CONFIG.CANVAS_HEIGHT - 280, 320, 120);
        // 车灯
        if (this.phase >= 1) {
            ctx.fillStyle = '#ff0';
            ctx.fillRect(this.carX + 300, CONFIG.CANVAS_HEIGHT - 260, 20, 20);
            ctx.fillRect(this.carX + 300, CONFIG.CANVAS_HEIGHT - 200, 20, 20);
        }

        // 天气效果

        }

        if (this.phase >= 2 && this.phase < 4) {

            ctx.fillRect(CONFIG.CANVAS_WIDTH / 2 - 300, 100, 600, 60);

        if (this.phase === 3) {

            ctx.fillText('AI智驾，守护每一次出行', CONFIG.CANVAS_WIDTH / 2, CONFIG.CANVAS_HEIGHT / 2 - 50);

                ctx.fillRect(0, 0, CONFIG.CANVAS_WIDTH, CONFIG.CANVAS_HEIGHT);
```

### 二、移动端适配方案

手机端支持是本游戏的重要需求，已在代码层面实现完整适配：

#### 2.1 强制横屏策略
- 竖屏握持时显示旋转提示界面（旋转手机图标动画 + "请横屏握持手机"文字）
- 检测 `orientation: portrait` 且屏幕宽度≤900px时自动显示提示层，遮挡游戏画面
- 使用 `@media (orientation: landscape)` 自动隐藏提示

#### 2.2 虚拟触控按钮
横屏状态下，触摸设备自动显示虚拟按钮：
- **左下角**：十字方向键（↑↓←→），直径60px圆形按钮（平板70px、小屏手机50px）
- **右下角**：动作按钮组
  - 绿色"跳"按钮（72px直径）= 空格跳跃
  - 橙色"蹲"按钮（72px直径）= C键下蹲
- 按钮使用半透明黑色背景+彩色边框，按压缩放反馈（scale 0.88 + 发光效果）
- 支持 `env(safe-area-inset-*)` 适配iPhone刘海屏/灵动岛安全区域
- 触摸事件同时映射到键盘输入系统，代码无需区分键盘/触摸

#### 2.3 响应式画布缩放
- 画布内部逻辑分辨率固定 **1920×1080**（16:9）
- 通过CSS自动缩放保持16:9比例：
  - 超宽屏幕（>16:9）：上下留黑边（letterbox），画布居中
  - 超窄屏幕（<16:9）：左右留黑边（pillarbox），画布居中
- `image-rendering: pixelated` 保证像素风画面在任意缩放下清晰锐利
- InputHandler自动将屏幕坐标转换为画布逻辑坐标，点击检测不受缩放影响

#### 2.4 性能分级（动态画质）
通过WebGL渲染器检测自动判定性能档位：
| 档位 | 设备类型 | 粒子数量 | 屏幕震动 | 阴影效果 |
|-----|---------|---------|---------|---------|
| high | PC/高端手机 | 200个（满效） | 启用 | 启用 |
| medium | 中端手机/平板 | 100个（50%） | 启用 | 禁用 |
| low | 低端手机 | 50个（25%） | 禁用 | 禁用 |

- 粒子发射数量通过 `getParticleRatio()` 自动缩放
- 低档位下禁用屏幕震动等非必要特效
- 可在运行时根据帧率动态降级

#### 2.5 触摸手势支持（额外）
- 点击画布 = 鼠标点击（菜单/按钮交互）
- 向上滑动 = 跳跃（可与虚拟按钮同时使用）
- 双指缩放 = 已阻止（防止误缩放页面）
- 长按 = 无特殊映射（避免误触）

### 三、图片尺寸自适应方案（解决贴图尺寸不符问题）

游戏不再依赖贴图必须符合规定尺寸，而是通过三重机制保证健壮性：

#### 3.1 assets_manifest.json 元数据声明
[assets_manifest.json](file:///c:/Users/ashin/Downloads/卓越工程师大冒险/assets/assets_manifest.json) 声明每个贴图的实际规格和目标渲染尺寸：

```json
{
  "characters": {
    "male_idle": {
      "file": "assets/images/characters/male_idle.png",
      "frameWidth": 96,
      "frameHeight": 96,
      "frameCount": 8,
      "targetWidth": 96,
      "targetHeight": 96
    }
  },
  "obstacles": {
    "dialog": {
      "pattern": "assets/images/obstacles/obstacle_dialog_{0}.png",
      "variants": 5,
      "targetWidth": 128,
      "targetHeight": 64
    }
  }
}
```

关键字段说明：
- `frameWidth/frameHeight/frameCount`：精灵图的帧尺寸和帧数（水平排列）
- `targetWidth/targetHeight`：游戏中实际渲染的目标尺寸（像素）
- `pattern`：变体文件命名模式，`{0}` 会被替换为01、02...
- `variants`：该类型的变体数量
- `sizes`：可选，每个变体的独立目标尺寸数组

#### 3.2 Canvas drawImage 自动缩放
AssetLoader的绘制方法自动处理尺寸适配：
```javascript
// 无论源图实际尺寸多大，都会绘制到 targetWidth × targetHeight 区域
ctx.drawImage(image, frameX, 0, frameW, frameH, dx, dy, targetW, targetH);
```
- 如果源图比目标尺寸大，自动缩小
- 如果源图比目标尺寸小，自动放大（像素风保持锐利）
- 精灵图帧的切割严格按 `frameWidth/Height`，不受源图实际比例影响

#### 3.3 占位图兜底（缺失贴图不崩溃）
如果某个贴图文件加载失败（不存在/路径错误/损坏）：
- **角色/道具/UI**：绘制红色虚线矩形 + 文字标签（显示资源key名），游戏继续运行
- **背景**：绘制渐变蓝色背景 + 网格线 + "[BG Placeholder]"标签
- **障碍物**：同样用红色占位框渲染，但碰撞逻辑正常运行
- 控制台会打印警告信息提示哪个资源缺失
- 游戏不会因为贴图缺失而白屏或崩溃，可以边开发边测试
